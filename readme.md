# 介绍

re_hcat-server(以下用"服务端"代替)是[HCat](https://hcat.online)的重置版服务器.

# 安装

目前,服务端暂未提供发行版,请使用git获取服务端.

```shell
git clone https://github.com/HCAT-Project/re_hcat-server.git
pip install -r requirements.txt
```

# 运行

```shell
python main.py
```
